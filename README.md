# qagenda

## Overview
qagenda is a TypeScript-based application designed to manage and organize your agenda efficiently.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/qagenda.git
   ```

2. Navigate into the project directory:
   ```
   cd qagenda
   ```

3. Install dependencies using Yarn:
   ```
   yarn install
   ```

## Development

To start the development server, run:
```
yarn dev
```

This command will compile the TypeScript code and run the application.

## Build

To build the project for production, use:
```
yarn build
```

This will compile the TypeScript files into JavaScript and output them to the `dist` directory.

## Usage

After building the project, you can run the compiled application from the `dist` directory.
